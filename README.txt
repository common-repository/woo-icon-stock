Ajoutez une icône de niveau de stock sur la fiche produit.

== Description ==

Ajoutez une icone de niveau de stock sur la fiche produit. 3 niveaux de stocks differents 'En stock, Stock faible, Hors stock'. 3 types d'icones au choix.

= Nouveau =
Maintenant, vous pouvez gerer les seuils de stock faible pour chaque article

== Configuration ==

Cliquez sur Woo Icon Stock. Changez les textes pour les articles en stock, hors stock et en quantite faible. Puis indiquez le seuil de la quantite faible par defaut.
Choisissez l'une des 3 icones ou uploader les votres.
Dans la page des produits allez à Woo Icon Stock puis cochez la case pour activer le seuil de stock faible pour le produit.


== Installation ==
Installez le plugin et cliquez sur activer. Cliquez sur l'onglet 'Woo Icon Stock' dans le back office et reglez les parametres.

1. Dezipez le fichier
2. Envoyez le fichier dans `wp-content/plugins/`
3. Activer le plugin sur la page des extensions
